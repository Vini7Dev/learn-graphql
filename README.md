# Scalar Types

**Text Types:**
- ID
- String

**Number Types:**
- Int
- Float

**Logical Types:**
- Boolean

**Array Types:**
- [Type]
  *Example: [Int]*

**Handling Response:**
- field: Type! (Not null)
  *Example: name: String!*
- field: [Type!]! (Not null and no elements null)
